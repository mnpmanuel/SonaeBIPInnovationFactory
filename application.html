Apply now

In order to apply, you have to:


1st - Download this application form file (click the icon below)



2nd - Fill in the file with all information regarding the project



3rd - Save the file



4th - Send your application using the link bellow (it will automatically open an email template containing a text sugestion to comply with the necessary legal requirements), or directly to sierrainnovation@sonaesierra.com

  • Include the application form as an email attach;
  • Include other attachments as complementary information to be considered for the application (not mandatory);
  • In the email body, the sender must declare its acceptance of the Sonae Sierra Innovation PowerUp Regulation;
  • In the email body, the sender must consent on the usage of personal data;
  • Include in cc all the team co-participants in the application.

    • All necessary steps in order to get consent for the usage of personal data, from other application co-participants, will be managed by email by the Innovation Office.
    Applications can only be accepted if the applicant (at least one), accepts the program regulation and provide consent for the usage of personal data as described in this same regulation.
    Any co-participant not providing acceptance of the regulation or consent on the usage of personal data (as described above), will not be considered as a valid applicant.
